# _Taproom App_

#### _Create imaginary inventory app for brewery_

#### By _**Ezra Sandzer-Bell**_

## Description

_This app lets the user see their inventory and details like cost per pint, quantity remaining, brand, and alcohol content by volume. Some css styling applied._

## Installation

_1. $ npm install_
_2. $ gulp serve_

## Features

* _App employs double binding to edit and update inventory in real time_
* _Pipe filter allows user to search inventory by brand name_
* _User can manipulate the dom on click (Pouring drinks function)_

## Technologies Used

_This site was built with Angular, NPM, Gulp, HTML, and CSS._
